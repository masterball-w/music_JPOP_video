# Demo 简介

**产品形态：** Python桌面应用/CLI工具，自动化生成日语歌词学习短视频

**面向用户：** JLPT N5-N2级别日语学习者、二次元文化爱好者、短视频内容创作者、日语教师

**核心功能：**

1. **智能歌曲获取** — 自动从内置194首热门日语歌曲库中获取歌曲信息，支持Spotify API扩展榜单
2. **日语歌词深度解析** — 基于MeCab形态分析，自动提取词汇（标注JLPT等级、读音、释义）和语法知识点（含例句）
3. **动漫风格视频生成** — 内置5套AI生成的动漫背景主题，歌词飞入飞出动画、解析卡片动态入场、Ken Burns背景缩放、粒子特效（樱花/光斑/星光）
4. **多平台格式适配** — 一键生成TikTok（1080x1920）、YouTube（1920x1080）、Instagram（1080x1080）三种格式

**已生成成果：** 70+首歌曲的学习短视频，涵盖YOASOBI、RADWIMPS、LiSA、Aimer、Ado等热门歌手

# Demo 创作思路

## 灵感来源

作为一名日语学习者和二次元文化爱好者，我发现自己在听日语歌曲时经常想了解歌词的含义和语法结构，但手动查词效率很低。同时，短视频平台上的日语学习内容质量参差不齐，制作成本高。如果能用AI自动生成高质量的歌词学习短视频，就能让更多人"边听歌边学日语"。

## 想解决的问题

日语学习者普遍面临三大痛点：

- **学习材料枯燥** — 传统教材内容单调，难以激发持续学习兴趣
- **词汇记忆困难** — 孤立记忆单词缺乏语境，容易遗忘
- **内容制作门槛高** — 手动制作学习视频需要设计、剪辑、排版等专业能力

## 为什么做这个方向

音乐是最佳的语言学习载体之一：歌词提供真实语境，旋律强化记忆关联，情感共鸣提升学习动力。而短视频是当下最主流的内容消费形式。将两者结合，用AI自动化生产流程，能同时解决"学习效率"和"内容供给"两个问题。

# 评审四维度说明

## 一、创新性（30%）

### 需求创新

传统日语学习工具（如多邻国、沪江）以教材为中心，缺乏与流行文化的结合。本项目首创"音乐驱动学习"模式，将被动听歌转化为主动学习——每首歌都成为一堂生动的语言课。

**差异化定位：**

| 对比维度 | 传统学习App | 本项目 |
|---------|-----------|--------|
| 内容来源 | 固定教材 | 实时热门歌曲榜单 |
| 学习形式 | 文字/选择题 | 沉浸式短视频 |
| 语境 | 人造例句 | 真实歌词语境 |
| 动力机制 | 打卡/积分 | 音乐情感共鸣 |
| 产出 | 无 | 可分享的短视频 |

### 解决思路创新

不依赖人工标注，而是通过MeCab日语形态分析引擎自动解析歌词，提取：
- **词汇层**：名词、动词、形容词、副词等，自动标注JLPT等级（N5-N1）
- **语法层**：识别语法模式（如「〜ように」「〜ている」），生成解释和例句
- **动态对齐**：歌词与解析内容实时对齐，每句歌词显示属于自己的解析卡片

当某行歌词没有预存解析时，系统会从该行的tokens中动态提取有价值的词汇（跳过助词和标点），确保每句歌词都有对应的解析内容。

### 技术创新

- **多层动画引擎** — 自研歌词飞入飞出动画系统（ease_out_back弹性入场、ease_in_cubic退场、sweep扫光高亮）
- **解析卡片交错入场** — 多张卡片以0.25秒间隔依次从不同方向飞入
- **Ken Burns背景效果** — 背景图片缓慢缩放平移，营造电影感
- **AI生成背景素材** — 使用AI生成5套动漫风格背景（星空湖泊、霓虹都市、神社黄昏、银河草原、夕阳田园）

# 开发心得

## 踩坑经历

1. **MoviePy 1.x vs 2.x API差异** — MoviePy 2.x将`set_audio`改为`with_audio`，`subclip`改为`subclipped`，需要做兼容处理
2. **日语字体渲染** — Windows下需要使用YuGothic或Meiryo字体才能正确显示日语字符
3. **歌词时间轴对齐** — LRC格式的时间戳精度不够，需要做毫秒级插值
4. **视频渲染性能** — 1080x1920@30fps的视频渲染较慢，通过限制max_duration和优化帧函数解决

## 解决方案

- 使用`getComputedStyle`从CSS变量读取颜色，确保图表与主题一致
- 实现`_extract_notes_from_tokens`方法，当行没有预存解析时从tokens动态提取
- 使用base64内嵌图片到HTML，确保作为附件上传时图片正常显示

## TRAE使用体验

TRAE IDE的AI辅助极大地加速了开发过程：
- **代码生成**：描述需求后，AI能快速生成完整的模块代码
- **问题诊断**：粘贴错误信息，AI能准确定位问题并提供修复方案
- **迭代优化**：通过多轮对话逐步优化视觉效果和修复bug
- **跨文件重构**：AI能理解项目结构，在多个文件间协调修改
